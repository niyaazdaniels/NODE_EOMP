Routes, Methods and Descriptions

Routes Method Description
Users
/ GET Current page
/users GET Retrieve all users' data
/users POST Add a user
/users/:id GET Retrieve a single user's data
/users/:id PATCH Update a user's data
/users/:id DELETE Delete a user's data
Products
/products GET Fetch all products from database
/products/:prodID GET Fetch a single product's data
/products/:prodID PATCH Update a product's data
/products/:prodID DELETE Delete a product
/products POST Add a product
